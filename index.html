Accessibility Bookmarklets

About

A bookmarklet is a type of bookmark, which instead of navigating to another web page, runs a snippet of JavaScript code on the currently loaded page in your browser.

These bookmarklets provide information about accessibility features that are included in the markup of web pages.

For example, drag the Headings link into your bookmarks toolbar and then click the link. You'll see that, if the page has HTML headings such as H1, H2, etc., they are highlighted with their heading level indicated.
